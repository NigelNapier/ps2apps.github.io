How to use this repository.


Install this repository and apply the changes automatically.

LOG INTO YOUR SERVER AS THE ROOT USER, sudo su is okay as well.
  1. Type this in your console to add this repository:

    2. zppy repo add repo.ps2apps.com

  2. Then type this in your console to update the zppy cache:

    3. zppy update

  3. Then install the module you require and wait for the process to finish:

    4. zppy install billing

    OR

    zppy install downloads

    OR

    zppy install mywebsql

  4. After the module is installed, navigate to Sentora's Module Admin and enable it:

  5. When there as an update available you need to do this:

    6. zppy update

  6. And then:

    7. zppy upgrade module_name


Install this repository manually.

Open Sentora and navigate to Module Admin. Scroll to the bottom and you will see a box to install a zppy file.
